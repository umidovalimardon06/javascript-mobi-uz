Дано число n. Необходимо написать функцию, которая проверяет, является ли это число палиндромом или нет.
Палиндром — это число, которое читается одинаково как слева направо, так и справа налево.

Примеры:
Палиндромы: 123321, 121, 111, 2222, 112211, 1, ...
Не палиндромы: 123, 112, 43, ...

Нужно выполнить с помощью массива. Нельзя использовать функцию Array.reverse и тип данных string (строка).
Значение n находится в следующем диапазоне:
0 <= n <= 10^18

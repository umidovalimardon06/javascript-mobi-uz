Написать стрелочную функцию которая принимает один параметр. Исходя из типа этой переменной функция должна возвращать результат:
- если это число нужно возвращать результат с квадратным корнем этого числа
- если это строка, тогда сначала нужно преобразовать его в число и потом нужно возвращать результат с квадратным корнем этого числа
- если это bool, то возвращать результат с текстом ошибки
- если это массив, тогда нужно вычислить квадратный корень каждого элемента и потом нужно возвращать результат с массивом
- если это объект, тогда нужно вычислить квадратный корень из суммы всех числовых полей и возвращать его

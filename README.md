Есть массив имен, фамилий, отчеств, стран. Все по 10 штук. Через функцию-конструктор нужно создать массив объектов спортсменов (20 штук) рандомно выбрав имя, фамилия, отчество, страну из списка и возраст (от 18-35 лет). А также должны быть поля "кол-во золотых медалей", "кол-во серебряных медалей", "кол-во бронзовых медалей" (по умолчанию значение 0). Нужно реализовать методы и функции:
- методы для увеличения и уменьшения кол-ва медалей;
- метод для копирования объекта;
- метод сравнения спортсмена с другим по кол-ву медалей;
- функция сортировки массива спортсменов по их рейтингу (олимпийский способ вычисления рейтинга из кол-ва медалей)

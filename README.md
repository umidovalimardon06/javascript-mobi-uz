Задача: "Таймер задач" — пошаговое выполнение

### Шаг 1. Создай менеджер задач с замыканием
Цель: попрактиковаться в замыкании и области видимости.

Условия:
Напиши функцию createTaskManager, которая возвращает объект с методами:
addTask(name, delay)
runAllTasks()

Все задачи должны храниться во внутреннем массиве, недоступном снаружи.
Используй let, не var.
Тема: замыкания, область видимости, let vs var
---

### Шаг 2. Реализуй метод addTask
Цель: потренироваться с остаточными параметрами и функциями.

Условия:
addTask(name, delay, ...tags) добавляет задачу в массив.
Каждая задача — объект с полями:
{
name: "Имя задачи",
delay: 2000, // миллисекунды
tags: ["тег1", "тег2"],
fn: function run() { ... } // функция выполнения
}

Используй именованное функциональное выражение (NFE) для fn.
Тема: остаточные параметры ..., объект функции (NFE)
---

### Шаг 3. Создай метод runAllTasks
Цель: освоить setTimeout, рекурсию и стрелочные функции.

Условия:
Метод runAllTasks(index = 0) запускает задачи по очереди, с задержкой.
Используй рекурсию: запусти задачу через setTimeout, затем вызови runAllTasks(index + 1).
Внутри setTimeout используй стрелочную функцию, чтобы не потерять this.

Тема: рекурсия, стрелочные функции, стек вызовов, setTimeout
---

### Шаг 4. Добавь вызов через call
Цель: научиться управлять контекстом.

Условия:
При запуске задачи вызывай её с контекстом задачи: task.fn.call(task)
Внутри run() можно будет писать, например:
console.log("Теги задачи:", this.tags);

Тема: call, контекст this
---

### Шаг 5. Привяжи функции к глобальному объекту
Цель: понять, что такое глобальный объект и как использовать bind.

Условия:
Сделай так, чтобы можно было вызывать addTask и runAllTasks из консоли браузера:
globalThis.addTask = manager.addTask;
globalThis.runAllTasks = manager.runAllTasks;

Тема: globalThis, область видимости
---

### Шаг 6. Продемонстрируй var и let
Цель: понять разницу между var и let в циклах и setTimeout.

Условия:
Напиши две петли:
в первой — for (var i = 0; i < 3; i++) + setTimeout(...)
во второй — for (let i = 0; i < 3; i++) + setTimeout(...)
Посмотри, как результат отличается.
Тема: var vs let, область видимости, асинхронность
---

### Шаг 7. Создай задачу с new Function
Цель: понять синтаксис new Function.

Условия:
Создай задачу, в которой функция создаётся динамически:
const dynamicFn = new Function("console.log('Динамическая задача')");
Добавь её в список задач и запусти вместе с другими.
Тема: new Function (edited)
